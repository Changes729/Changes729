# ISO 15693

如果说 ISO 14443 的 Mifare 卡类似U盘的话，那么 ISO 15693 的 ICode 卡就更类似一个应用程序。

与 Mifare 卡交互时，比较像读写文件。但和 ICode 卡交互时，就会发送许多指令。
