## 电池类型

- 根据储能材料分类
  - 磷酸铁锂-锂离子电池
  - 碳锌锰干电池（碱性电池、一次性电池）
- 根据形状来分
  - 圆柱电池
  - 方形电池
  - 纽扣电池



18650锂电池可充电，其外径18mm，长度65。和AA电池（5号电池，外径14.5mm，长度50,14500）、AAA电池（7号电池，10.5外径，44.5长度，10440）都不一样
