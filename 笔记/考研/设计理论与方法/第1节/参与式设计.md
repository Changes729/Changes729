# 概念

指设计团队在开发过程中，邀请利益相关者各方参与，一同定义问题、定位产品、提出解决方案，并且参与方案评估.

# 过程

1. 在社区内寻找相关人员。
2. 让大伙儿理解当前社区需要做的时期。
3. 将大伙儿的想法整理归纳为可能的解决方案。测试并且优化原型方案。
4. 交付，大伙儿一同设计的产品，往往会得到大伙儿的关心和热情，这确定了这样有效的解决方案时可持续的。

# 方法

- 焦点小组
- 一对一
- 在线开放式（众包）
