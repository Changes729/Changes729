# 手绘设计细节

- **线条的自然、流畅才是重点**
- 线条要干净，利落。
- 线条方向（自由直线）练习，45 度、135 度、90 度和 0 度。一般右利手都是从左向右画。上下是从下往上画。
- 直线之间的距离需要把握。
