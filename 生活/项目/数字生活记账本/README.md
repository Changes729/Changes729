# 数字时代个人生活记账本

现代资本社会主要以**资金**为主要生存手段，对资金进行管控是高质量生活的一种态度与方式。

当下，记账本的主要问题是<u>记账本手动记录太麻烦</u>。我们希望通过某种方式，使我们的经济生活达到一种**收支既记录**的愿景。
