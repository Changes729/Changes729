# 微波技术和天线

**辐射功率**

1. 电流I越大，辐射功率越大
2. 振子的电长度$\Delta{l} /\lambda $ 越大，辐射功率越大（恒流源情况下）
3. 辐射功率与距离r无关



 **辐射电阻**

$R_r = 80 \pi^2(\frac{\Delta{l}}{\lambda})^2$
