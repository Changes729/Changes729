# 贴片器件类型

- QFN（quad flat no-lead package，对四角扁平无铅封装）
- LQFP（low profile quad flat package，低剖面四平封装）
- BGA（Ball grid array，球栅阵列）
- CSP（Chip-scale package，芯片级封装）
