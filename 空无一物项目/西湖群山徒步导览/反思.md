# 反思

## 反思办法

**01 个人反思**：（仅供参考，无强制要求）

项目总览（感知分析）

1. 个人目标明确
2. 个人成果记录
3. 不足点记录
4. 感知分析（感觉奇怪、不对的时刻；感觉好的、激动的时刻等）

资源投入分析（数据分析）

1. 时间投入
2. 资源投入
3. 其他投入

总结记录

1. 首先总结不足与理想目标的差距
2. 其次总结项目过程中的成果（任何认为是成果的成果）



**02 团队反思**：

团队合作

1. 从其他成员中获取的、学到的内容（夸夸）
2. 项目进程中的明显不足（批评）
3. 对不足的可能解决方案
4. 对成果的总结和再学习（团队内分享、项目后技能教学、发文形成社会影响力）
